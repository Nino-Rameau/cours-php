# Initiation PHP
Projet simple pour découvrir les bases du langage PHP.

## Objectif
Apprendre les fondamentaux :

- syntaxe PHP
- variables
- conditions
- boucles
- fonctions

## Prérequis
- PHP 8+
- serveur local (XAMPP, WAMP, MAMP...)
- navigateur web

## Installation
Cloner le repository :

```bash
git clone https://github.com/username/cours-php.git
```

Entrer dans le dossier :

```bash
cd cours-php
```

Placer le projet dans le dossier de votre serveur local puis ouvrir dans le navigateur :

```
http://localhost/cours-php
```
